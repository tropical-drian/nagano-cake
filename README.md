## 管理者側
1. メールアドレスとパスワードでログイン※新規登録不可
2. 商品の新規追加、詳細編集、販売ステータスの切り替え
3. 商品検索機能（商品名・商品説明・商品ID・販売ステータス）
4. ジャンルの新規追加・ジャンル名変更
5. 会員情報の閲覧、編集、退会状態への切り替え
6. 注文ステータスと商品の製作ステータスの切り替え
